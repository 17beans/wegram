# wegram 앱은...

Instagram 앱과 비슷한 기능을 수행하는 앱이다.

# 사용 기술 스택

- React Native With Expo (expo-cli)
- Firebase Authentication
- Firebase Cloud Firestore

# 기능 및 UI 구현

기존 인스타그램 보다 가볍게 기능을 구현하였으며, 심플한 디자인을 추구했고 메인 페이지 상단에 간단한 광고를 보일 수 있도록 구성함
